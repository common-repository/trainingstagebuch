Zeigt Trainigsdaten in deinem Blog an.

== Description ==

Zeigt Trainigsdaten von [Trainingstagebuch.org](http://www.trainingstagebuch.org/) in deinem Blog an:

* nutze das [Widget](http://www.christianschenk.org/de/projects/wordpress-trainingstagebuch-plugin/#widget), um verschiedene Banner, Badges oder Informationen zu deinen monatlichen oder jählichen Einheitung anzuzeigen
* platziere den [Shortcode](http://www.christianschenk.org/de/projects/wordpress-trainingstagebuch-plugin/#shortcode) in deinen Beiträgen wenn du deine Trainingsdaten in einen Beitrag einbinden möchtest.

== Installation ==

Eine manuelle Installation geht so:

1. Plugin herunterladen, im Verzeichnis wp-content/plugins entpacken und im Admin-Bereich aktivieren.
2. [Widget](http://www.christianschenk.org/de/projects/wordpress-trainingstagebuch-plugin/#widget) verwenden oder
3. [Shortcode](http://www.christianschenk.org/de/projects/wordpress-trainingstagebuch-plugin/#shortcode) nutzen.
